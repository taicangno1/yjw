# 真机调试指南

## 环境准备

1. 安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 登录微信开发者账号
3. 手机开启开发者模式（设置 → 关于手机 → 连续点击版本号7次）

## 项目导入

1. 打开微信开发者工具
2. 选择「导入项目」
3. 选择 `what-to-eat` 文件夹
4. AppID 输入你的小程序 AppID（如无，可选「体验版」）
5. 点击「确认」

## 真机调试步骤

### 1. 模拟器调试
- 点击「编译」按钮可在模拟器中预览
- 使用「热重载」自动更新代码

### 2. 真机调试
- 点击「真机调试」按钮
- 扫描二维码连接手机
- 在 Chrome DevTools 中调试 WXML/CSS/JS

### 3. 体验版预览
- 点击「上传」上传代码
- 在微信公众平台设置体验版
- 扫码体验

## 调试面板功能

| 面板 | 功能 |
|------|------|
| WXML | 查看和编辑页面结构 |
| Console | 查看日志和报错 |
| Sources | 断点调试 JS |
| Network | 查看网络请求 |
| Storage | 查看本地存储 |

## 常见问题

### 定位权限不生效
- 确保在「详情」→「调试」→「不校验合法域名」已勾选
- 手机设置中允许微信使用定位

### 图片不显示
- 检查图片 URL 是否可访问
- 开发模式下可暂时关闭域名校验

### 跳转美团失败
- 检查 `app.json` 中的 `lazyCodeLoad` 配置
- 确保已添加美团小程序 appId 白名单
