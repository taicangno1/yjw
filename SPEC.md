# 名字大宗师 - 起名软件规格文档

## 1. 项目概述

- **项目名称**: 名字大宗师
- **项目类型**: H5 起名应用
- **核心功能**: 基于周易八字五行分析，智能推荐吉祥名字
- **目标用户**: 为新生儿起名的父母

## 2. 功能模块

### 2.1 八字五行分析
- 输入：出生年月日时（阳历/阴历）
- 输出：八字命盘、五行分布、强弱分析
- 喜用神推荐

### 2.2 周易卦象
- 基于姓名笔画数起卦
- 64卦卦象解析
- 卦辞/彖辞/象辞解释

### 2.3 三才五格
- 天格、地格、人格、外格、总格
- 五格数理吉凶
- 姓名综合评分

### 2.4 名字推荐
- 根据五行缺失推荐
- 结合生肖宜用字
- 音韵美感分析

## 3. 技术实现

- 纯前端 H5 单页应用
- HTML5 + CSS3 + Vanilla JavaScript
- 无需后端，纯本地计算

## 4. 验收标准

- [x] 可输入出生信息进行八字分析
- [x] 可计算三才五格
- [x] 可获取名字推荐
- [x] 界面美观，操作流畅
